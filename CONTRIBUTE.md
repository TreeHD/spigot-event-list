# Contribute

## コマンド

### イベント内容の更新
#### Javadoc の内容を適応
```shell
npm run download
```

#### events.yaml の変更を適応
```shell
npm run generate
```

### 依存関係の更新
```shell
npm run ncu:update
```

## コミットメッセージテンプレート

### バージョンの更新
```
ライブラリのバージョンを更新
```
